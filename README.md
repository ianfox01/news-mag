## Story/Feature Template

<table>
<thead>
  <tr>
    <th>Class</th>

  </tr>

<tbody>
  <tr>
    <td>.body-container   </td>
    <td>default   </td>
    <td>The default layout is a single, centered   column. It has no additional class declaration.   </td>
  </tr>
  <tr>
    <td>.body-container.two-col   </td>
    <td>optional   </td>

## Hero Options

<table>
<thead>
  <tr>
    <th>Class</th>

    <td>optional   </td>

  </tr>
  <tr>
    <td>.hero.simple</td>
    <td>optional   </td>
    <td>No image.</td>
  </tr>
